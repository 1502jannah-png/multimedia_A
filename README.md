Galeri Multimedia (Static HTML)
===============================

Cara menggunakan:
1. Buka folder ini di komputer dan jalankan `index.html` di browser modern.
2. Tambahkan file media (jpg/png/mp4/mp3) ke folder `media/`.
3. Edit `data/media.json` untuk menambahkan entri media. Contoh entri:
   {
     "id": 4,
     "title": "Judul Baru",
     "type": "image",
     "file": "media/yourfile.jpg",
     "description": "Deskripsi",
     "category": "Fotografi",
     "approved": true
   }

Catatan:
- Versi ini adalah situs statis; fitur upload atau admin dinamis membutuhkan backend.
- Tema warna biru-hitam sudah diterapkan di assets/style.css.
